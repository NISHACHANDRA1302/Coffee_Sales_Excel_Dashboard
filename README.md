# ☕ Coffee Sales Analysis Dashboard (Excel)

## 📌 Project Overview
This project focuses on analyzing coffee shop sales data using Excel to uncover key business insights. The dashboard provides a clear and interactive view of sales performance, customer behavior, and product trends.

The goal is to help stakeholders make data-driven decisions by identifying high-performing products, peak sales times, and store-level performance.

---

## 🎯 Problem Statement
The coffee shop generates large volumes of sales data daily, but lacks a structured way to:
- Track overall sales performance
- Identify top-selling products
- Analyze customer purchasing patterns
- Monitor store-wise performance

This project solves these challenges by creating an interactive Excel dashboard.

---

## 🛠️ Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning & Transformation
- Dashboard Design

---

## 📊 Key KPIs
- Total Sales Revenue
- Total Orders / Transactions
- Total Quantity Sold
- Average Sales per Transaction
- Category-wise Sales
- Store-wise Performance

---

## 📈 Dashboard Features
- 📅 Sales trends over time (daily/monthly)
- 🏪 Store location-wise performance
- ☕ Product category and type analysis
- ⏰ Peak hours analysis (based on transaction time)
- 📊 Interactive filters (slicers)

---

## 🔍 Key Insights
- Certain product categories contribute the highest revenue
- Sales peak during specific hours of the day
- Some store locations perform better than others
- High-demand products can be identified for inventory planning

---

## 📁 Dataset Description
The dataset includes the following columns:
- Transaction ID
- Transaction Date & Time
- Quantity Sold
- Store Location
- Product Category & Type
- Unit Price
